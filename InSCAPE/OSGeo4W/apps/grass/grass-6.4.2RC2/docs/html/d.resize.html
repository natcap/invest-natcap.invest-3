NAME

d.resize - Resizes active display monitor

KEYWORDS

display, setup

SYNOPSIS

d.resize
d.resize help
d.resize width=integer height=integer [--verbose] [--quiet]

Parameters:

width=integer
New width for window
height=integer
New height for window

DESCRIPTION

d.resize Sets the current display monitor window size to specified height and width.
