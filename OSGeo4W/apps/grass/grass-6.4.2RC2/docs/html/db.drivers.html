NAME

db.drivers - List all database drivers.

KEYWORDS

database, attribute table

SYNOPSIS

db.drivers
db.drivers help
db.drivers [-fp] [--verbose] [--quiet]

Flags:

-f
Full output
-p
print drivers and exit
--verbose
Verbose module output
--quiet
Quiet module output

DESCRIPTION

db.drivers lists all database drivers.
