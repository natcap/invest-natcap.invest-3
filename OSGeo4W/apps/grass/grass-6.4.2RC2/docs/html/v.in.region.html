NAME

v.in.region - Create a new vector from the current region.

KEYWORDS

vector, geometry

SYNOPSIS

v.in.region
v.in.region help
v.in.region output=name [type=string] [cat=integer] [--overwrite] [--verbose] [--quiet]

Flags:

--overwrite
Allow output files to overwrite existing files
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

output=name
Name for output vector map
type=string
Select type: line or area
Options: line,area
Default: area
cat=integer
Category value
Default: 1

DESCRIPTION

v.in.region creates a new vector map from current region.
