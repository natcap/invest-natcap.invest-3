NAME

v.build.all - Rebuilds topology on all vector maps in the current mapset.

KEYWORDS

vector

SYNOPSIS

v.build.all
v.build.all help
v.build.all [--verbose] [--quiet]

DESCRIPTION

v.build.all runs v.build for all vectors in the current mapset.
